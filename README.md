# 動画字幕（テロップ）自動生成ツール

動画を読み込ませると、音声を AI（Whisper）で文字起こしして字幕を自動でつけます。
**完全ローカル動作**（インターネット接続はモデル初回ダウンロード時のみ・APIキー不要・動画は外部に送信されません）。

## できること
- 動画 → 字幕ファイル（`.srt`）を自動生成
- （オプション）動画にテロップを焼き込んだ動画を出力

## 使い方

### かんたん（ドラッグ&ドロップ）
1. 動画ファイルを **`字幕をつける.bat`** のアイコンにドラッグ＆ドロップ
2. 同じフォルダに `元の動画名.srt` ができます

### コマンドで使う
```powershell
# 字幕ファイル(.srt)だけ作る
python make_subtitle.py "動画.mp4"

# 字幕を焼き込んだ動画も作る（動画名_字幕付き.mp4 ができる）
python make_subtitle.py "動画.mp4" --burn

# 精度を上げる（遅くなります）
python make_subtitle.py "動画.mp4" --model large-v3

# 英語の動画
python make_subtitle.py "動画.mp4" --lang en
```

## 生成された .srt の使い方
- **YouTube**: 動画アップ時に字幕ファイルとして `.srt` をそのまま読み込めます
- **動画編集ソフト / プレーヤー**: 動画と同じ名前の `.srt` を同じフォルダに置くと自動表示されます（VLC など）
- 中身はテキストなので、誤認識があればメモ帳などで直接修正できます

## モデルの選び方（精度↑＝速度↓）
| モデル | 精度 | 速度 | 用途 |
|--------|------|------|------|
| tiny / base | 低 | 速い | お試し |
| small | 中 | 普通 | |
| **medium**（既定） | 高 | やや遅い | 日本語の推奨ライン |
| large-v3 | 最高 | 遅い | 仕上げ・重要な動画 |

初回実行時に選んだモデルが自動ダウンロードされます（medium で約1.5GB）。

## 必要環境（セットアップ済み）
- Python 3.14
- `faster-whisper`（`pip install faster-whisper` でインストール済み）
- ffmpeg（`--burn` で焼き込む場合のみ必要。インストール済み）

## 調整したいとき
`make_subtitle.py` 冒頭の設定で変更できます:
- `DEFAULT_MODEL` … 既定のモデル
- `MAX_CHARS` … テロップ1枚あたりの最大文字数
- `MAX_DURATION` … テロップ1枚あたりの最大秒数
- `SUBTITLE_STYLE` … 焼き込み時の文字の大きさ・色・位置
